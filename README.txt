Tower Scaler (Platformer)
by Markus Feng (assisstion)

This is an arcade game where the objective is to climb a down-scrolling tower while staying alive as long as you can by jumping from platform to platform. This game is still in development mode - this is a work in progress, and the completion date is not known yet.
